== Apache Daffodil (incubating) Scala API ==

=== Packages ===

[[edu.illinois.ncsa.daffodil.sapi]] - Provides the classes necessary to compile DFDL schemas, parse and unparse files using the compiled objects, and retrieve results and parsing diagnostics

[[edu.illinois.ncsa.daffodil.sapi.logger]] - Provides the classes necessary to recieve logging messages from Daffodil.

[[edu.illinois.ncsa.daffodil.sapi.debugger]] - Provides the classes necessary to perform parse tracing or create a custom debugger
